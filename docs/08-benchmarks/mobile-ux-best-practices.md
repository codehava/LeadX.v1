## ğŸ¯ Core Principles

### 1. Mobile-First, Not Mobile-Friendly

| Principle | Implementation |
|-----------|----------------|
| **Design for thumb** | Put primary actions in thumb-friendly zones |
| **Minimal input** | Smart defaults, autofill, suggestions |
| **Fast** | Local-first, instant feedback |
| **Contextual** | Show relevant info at right time |
| **Glanceable** | Key info visible at a glance |
