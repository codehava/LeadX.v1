## ðŸ—„ï¸ Database Schema

> **Note:** Uses a **combined table approach** where `cadence_participants` contains
> attendance, form submission, and feedback data. See [cadence.md](../../04-database/tables/cadence.md)
> for full schema details.

### `cadence_participants` Table (Combined)

```sql
CREATE TABLE cadence_participants (
  id UUID PRIMARY KEY,
  meeting_id UUID REFERENCES cadence_meetings(id),
  user_id UUID REFERENCES users(id),

  -- ATTENDANCE (marked during meeting by host)
  attendance_status VARCHAR(20),      -- PENDING, PRESENT, LATE, EXCUSED, ABSENT
  arrived_at TIMESTAMPTZ,
  excused_reason TEXT,
  attendance_score_impact INTEGER,    -- +3 present, +1 late, 0 excused, -5 absent
  marked_by UUID,
  marked_at TIMESTAMPTZ,

  -- PRE-MEETING FORM (Q1-Q4)
  pre_meeting_submitted BOOLEAN,
  q1_previous_commitment TEXT,        -- Auto-filled from last Q4
  q1_completion_status VARCHAR(20),   -- COMPLETED, PARTIAL, NOT_DONE
  q2_what_achieved TEXT,              -- Required
  q3_obstacles TEXT,                  -- Optional
  q4_next_commitment TEXT,            -- Required
  form_submitted_at TIMESTAMPTZ,
  form_submission_status VARCHAR(20), -- ON_TIME, LATE, VERY_LATE, NOT_SUBMITTED
  form_score_impact INTEGER,          -- +2 on-time, 0 late, -1 very late, -3 not submitted

  -- HOST NOTES & FEEDBACK
  host_notes TEXT,                    -- Internal notes (not visible to participant)
  feedback_text TEXT,                 -- Formal feedback visible to participant
  feedback_given_at TIMESTAMPTZ,
  feedback_updated_at TIMESTAMPTZ,

  -- SYNC
  last_sync_at TIMESTAMPTZ,

  UNIQUE(meeting_id, user_id)
);
```

### Field Descriptions

| Field | Purpose | Visible To | When Set |
|-------|---------|------------|----------|
| `host_notes` | Quick notes during meeting | Host only | During meeting |
| `feedback_text` | Formal feedback for participant | Host + Participant | During/after meeting |
| `feedback_given_at` | When feedback first submitted | System | First save |
| `feedback_updated_at` | When feedback last edited | System | Each edit |
| `attendance_score_impact` | Points for attendance | Participant | When marked |
| `form_score_impact` | Points for form submission | Participant | On submission |
